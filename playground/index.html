Open your browser's developer console (F12 or Cmd+Option+I) and start experimenting!

Available global variables:

  • storage - Wrapped localStorage with 60 seconds expiry
  • wrapStorage - The wrapStorage function to create your own instances

Try these commands:

// Set a value that expires in 60 seconds storage.setItem("user", "John Doe") // Get the value storage.getItem("user") // "John Doe" // Wait 60+ seconds and try again storage.getItem("user") // null (expired) // Create your own storage with different expiry const quickStorage = wrapStorage(localStorage, { expiresInSeconds: 10 }) quickStorage.setItem("temp", "expires quickly") // Other methods storage.removeItem("user") storage.clear() storage.length storage.key(0)
